# Dog Breed Weighted Scoring Matrix

This matrix evaluates dog breeds for an active couple living in an urban Mid-Atlantic region with access to green space, light hiking, medium size preference, social personality, low-maintenance grooming, and apartment-friendly living.

## Step 1: Define Criteria and Weights

| Criterion | Weight (%) | Description |
|-----------|------------|-------------|
| Size | 15 | Medium size (not too big or small) |
| City Adaptability | 15 | Comfortable living in apartment/city environment |
| Activity / Hike Friendliness | 20 | Can handle light hikes and outdoor activity |
| Temperament / Personality | 20 | Social, friendly, engaging |
| Grooming / Care | 15 | Low-maintenance grooming (shedding acceptable) |
| Social / Companionship | 10 | Bonds well, enjoys human interaction |
| Climate Tolerance | 5 | Comfortable in Mid-Atlantic weather (hot summers, humid, mild winters) |

Weights total = 100%
